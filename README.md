## 🎯 Controle de Níveis de Água

- Programa simples e eficiente de monitoramento de um reservatório de água, que será executado no terminal e tem como objetivo exibir mensagens de alerta com cores diferentes, conforme o nível de água.

## 📝 Sobre o Monitoramento
- Trabalharemos com cinco níveis representados em uma lista, do nível 1 (mínimo) até o nível 5 (máximo) do reservatório.
- Utilizaremos a biblioteca colorama para tornar a visualização mais clara, destacar cada situação do Nível com uma cor específica, através da semântica de cores (quentes e frias) e definidas para cada nível.


## 🚀 Resultado
- Cada nível foi associado a uma mensagem de alerta e com a sua respectiva cor, seguindo a progressão de risco, assim como a transição das cores com tons frios para os tons quentes.
